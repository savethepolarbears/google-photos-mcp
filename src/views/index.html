Google Photos MCP Server

Model Context Protocol (MCP) server for Google Photos integration

Server Running

Server Endpoints

MCP Endpoint: /mcp - Use this in Claude Desktop or Cursor IDE
Authentication: /auth - Authenticate with Google Photos
Health Check: /health - Server health status
Authenticate with Google Photos

Available Tools

  • search_photos - Search photos by text query
  • search_photos_by_location - Search photos by location
  • get_photo - Get a specific photo by ID
  • get_photo_url - Get the URL of a specific photo
  • list_albums - List all albums
  • get_album - Get a specific album by ID
  • list_album_photos - List photos in an album

Setup Instructions

Claude Desktop

  1. Start the server in STDIO mode: npm run stdio
  2. In Claude Desktop, go to Settings > MCP Servers
  3. Add a new server and select "Local Command"
  4. Enter the command: node /Users/klkro/claude-mcps/google-photos/dist/index.js --stdio
  5. Name it "Google Photos"

Cursor IDE

  1. Start the server in HTTP mode: npm start
  2. In Cursor IDE, configure the MCP server
  3. Add the server URL: http://localhost:3000/mcp
  4. Name it "Google Photos"
